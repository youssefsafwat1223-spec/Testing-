Project Introduction

Our project is REAL TIME IoT DATA SENSORS.

AquaSense aligns with Egypt's 2030 Vision by providing a real-time monitoring system for Egyptian fish farms. It supports up to 11 tanks (multiple species) with 8 sensors per tank. The system is expandable: more tanks, more sensors, more intelligence.

AquaSense contributes to Egypt's sustainability goals as emphasized by national leadership.

1. Real-time Data Simulation

We created a realistic dataset by merging two real fish-farm datasets. No random data was used. Columns include: fish, temperature, humidity, light, oh, electricalconductivity, turbidity, watertemp, tds.

2. ETL System

A Python ETL script runs every 5 seconds: Extracts data → Classifies readings → Loads to SQLite database.

3. Alert System

The Kafka alert system triggers warnings when classification enters F+, F++, F-, or F--. A Python worker checks values every 5 seconds.

4. Real-time Dashboard

The final stage is a Flask dashboard showing trends, alerts, and the latest sensor values.
